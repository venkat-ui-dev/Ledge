# Ledge — Landing Page

A modern, responsive SaaS landing page built with **Next.js**, **React**, and **Tailwind CSS**, designed and developed as a portfolio piece.

**Concept:** Ledge is a time-tracking and invoicing tool for freelance developers and designers — the hero section features a live "running invoice" ticker that counts up time and earnings in real time.

## Sections

- Hero with animated live-invoice signature element
- Features (4 cards)
- Pricing (3 tiers, one highlighted)
- Contact form (client-side only — no backend wired up)
- Footer

## Tech stack

- Next.js 14 (Pages Router)
- React 18
- Tailwind CSS
- lucide-react (icons)

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000
